## 🚀 Featured Projects

### 🛡️ [APSIT S.A.F.E](https://github.com/AyyIM24/APSIT-S.A.F.E)
> *Student Asset Finder & Escort — Campus Lost & Found, reimagined*

A full-stack web platform that digitizes lost & found management for university campuses. Built with a **dual-portal system** (Student + Admin), it features QR-code-based item verification, JWT-secured claim submission, and a real-time admin analytics dashboard.

**Stack:** React 18 · Framer Motion · Recharts · Spring Boot · Spring Security · MySQL · ZXing (QR)

🔗 [Live Demo](https://apsit-s-a-f-e.vercel.app)
